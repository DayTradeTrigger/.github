# about TradeTrigger

기능
1. AP, Reuters, Bloomberg와 Yahoo Finance, Benzinga를 크롤링하여 세계 정세 파악
2. 연합뉴스, 연합인포맥스 크롤링하여 국내 정세 파악
3. 주가 변동성을 가져올 정도의 핵심 이벤트가 있을 때 테마주 판단
4. 테마 내에서 재무건전성이 안정적이고, 주식의 누름세가 오직 해당 이벤트로 인해서만인 종목 판단
5. 해당 이벤트가 지속될 때의 가능한 테마주와 완료되었을 때의 가능한 테마주 판단
6. 해당 테마주 내에서 4번의 조건을 만족하는 종목을 예비 대장주로써 판단하여 보여줌

ai 설정
1. 각각의 기능을 세분화하여 모델 객체를 생성
2. chatgpt api 사용할 예정 (groq 씹쓰레기임/RAG가 아니라 그런 듯)
